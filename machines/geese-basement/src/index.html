artifacts of geese from the basement

these ten images have sound waves inside of them. they are frames from this video, with audio from the same moment encoded into their pixels. select one and listen to it loop.

tools provided by another machine.

The visual distortion in these images is the result of an audio waveform stored in image pixels. They were created using code libraries that can be found at amplib.app. The same tools are used here to decode and play them back as sound.

There are no audio files, just the images you see here. I claim no rights to the original works, this sampling is meant to direct attention to the artists.
